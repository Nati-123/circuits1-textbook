# Chapter 3: Kirchhoff’s Laws and Circuit Topology

## 3.1 Kirchhoff’s Current Law (KCL)

KCL states that the sum of currents entering a node is equal to the sum of currents leaving the node.

\[
\sum I_{in} = \sum I_{out}
\]

## 3.2 Kirchhoff’s Voltage Law (KVL)

KVL states that the sum of voltages around any closed loop is zero.

\[
\sum V = 0
\]

## 3.3 Example

Consider a simple loop with a voltage source and resistors.

Using KVL:
\[
V - IR_1 - IR_2 = 0
\]

## 3.4 Circuit Topology

Circuit topology describes how elements are connected without considering values.

- Nodes
- Branches
- Loops

## 3.5 Summary

- KCL → currents at node  
- KVL → voltages in loop  
- Used to solve circuits systematically
