<h1 style="color: #5A3EED !important; border-bottom: 3px solid #5A3EED; padding-bottom: 0.4rem; font-weight: 800; margin-bottom: 1.5rem;">Glossary: Kirchhoff's Laws and Circuit Topology</h1>

<p style="color: #555; line-height: 1.85; font-size: 1.05rem; margin-bottom: 2rem;">
Key terms and definitions for Chapter 3. Definitions follow ISO 11179 metadata registry standards — precise, concise, distinct, and non-circular.
</p>

<div style="background: #EEF4FF; border: 2px solid #A8C8FF; border-radius: 12px; padding: 28px 28px 12px 28px; margin: 1.5rem 0; box-shadow: 0 2px 8px rgba(90,61,237,0.07);">

<p style="color: #1565C0; font-weight: 700; font-size: 1.08rem; margin-top: 0; margin-bottom: 18px;">B</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;">
<strong style="color: #5A3EED;">Branch</strong> — A path containing a single circuit element that connects two nodes. Each branch carries its own current.
</p>

</div>

<div style="background: #EEF4FF; border: 2px solid #A8C8FF; border-radius: 12px; padding: 28px 28px 12px 28px; margin: 1.5rem 0; box-shadow: 0 2px 8px rgba(90,61,237,0.07);">

<p style="color: #1565C0; font-weight: 700; font-size: 1.08rem; margin-top: 0; margin-bottom: 18px;">C</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;">
<strong style="color: #5A3EED;">Circuit Simplification</strong> — The process of reducing a complex circuit to a simpler equivalent form using series, parallel, and delta-wye techniques while preserving electrical behavior at the terminals.
</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;">
<strong style="color: #5A3EED;">Circuit Topology</strong> — The study of how circuit components are interconnected, independent of component types or values, described in terms of nodes, branches, loops, and meshes.
</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;">
<strong style="color: #5A3EED;">Conductance Matrix</strong> — A symmetric matrix \(\mathbf{G}\) used in the node voltage method where diagonal entries are the sum of conductances connected to each node, and off-diagonal entries are the negatives of shared conductances.
</p>

</div>

<div style="background: #EEF4FF; border: 2px solid #A8C8FF; border-radius: 12px; padding: 28px 28px 12px 28px; margin: 1.5rem 0; box-shadow: 0 2px 8px rgba(90,61,237,0.07);">

<p style="color: #1565C0; font-weight: 700; font-size: 1.08rem; margin-top: 0; margin-bottom: 18px;">D</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;">
<strong style="color: #5A3EED;">Delta Configuration (Δ)</strong> — A three-terminal resistor arrangement where three resistors are connected in a triangle, with each resistor directly linking two of the three terminals. Also called a pi (π) configuration.
</p>

</div>

<div style="background: #EEF4FF; border: 2px solid #A8C8FF; border-radius: 12px; padding: 28px 28px 12px 28px; margin: 1.5rem 0; box-shadow: 0 2px 8px rgba(90,61,237,0.07);">

<p style="color: #1565C0; font-weight: 700; font-size: 1.08rem; margin-top: 0; margin-bottom: 18px;">E</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;">
<strong style="color: #5A3EED;">Equivalent Resistance</strong> — A single resistance value that replaces a network of resistors while drawing the same total current from the source as the original network.
</p>

</div>

<div style="background: #EEF4FF; border: 2px solid #A8C8FF; border-radius: 12px; padding: 28px 28px 12px 28px; margin: 1.5rem 0; box-shadow: 0 2px 8px rgba(90,61,237,0.07);">

<p style="color: #1565C0; font-weight: 700; font-size: 1.08rem; margin-top: 0; margin-bottom: 18px;">K</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;">
<strong style="color: #5A3EED;">Kirchhoff's Current Law (KCL)</strong> — The principle that the algebraic sum of all currents at any node equals zero, based on conservation of electric charge.
</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;">
<strong style="color: #5A3EED;">Kirchhoff's Voltage Law (KVL)</strong> — The principle that the algebraic sum of all voltages around any closed loop equals zero, based on conservation of energy.
</p>

</div>

<div style="background: #EEF4FF; border: 2px solid #A8C8FF; border-radius: 12px; padding: 28px 28px 12px 28px; margin: 1.5rem 0; box-shadow: 0 2px 8px rgba(90,61,237,0.07);">

<p style="color: #1565C0; font-weight: 700; font-size: 1.08rem; margin-top: 0; margin-bottom: 18px;">L</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;">
<strong style="color: #5A3EED;">Load Resistance</strong> — The resistance of the component or subsystem that receives and consumes power from the circuit.
</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;">
<strong style="color: #5A3EED;">Loop</strong> — Any closed path through a circuit that starts and ends at the same node without passing through any node more than once.
</p>

</div>

<div style="background: #EEF4FF; border: 2px solid #A8C8FF; border-radius: 12px; padding: 28px 28px 12px 28px; margin: 1.5rem 0; box-shadow: 0 2px 8px rgba(90,61,237,0.07);">

<p style="color: #1565C0; font-weight: 700; font-size: 1.08rem; margin-top: 0; margin-bottom: 18px;">M</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;">
<strong style="color: #5A3EED;">Mesh</strong> — A loop that does not enclose any other loops within it; the smallest possible loop in a planar circuit.
</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;">
<strong style="color: #5A3EED;">Mesh Current Method</strong> — A systematic circuit analysis technique that assigns a circulating current to each mesh and writes KVL equations, requiring \(m\) equations for \(m\) meshes.
</p>

</div>

<div style="background: #EEF4FF; border: 2px solid #A8C8FF; border-radius: 12px; padding: 28px 28px 12px 28px; margin: 1.5rem 0; box-shadow: 0 2px 8px rgba(90,61,237,0.07);">

<p style="color: #1565C0; font-weight: 700; font-size: 1.08rem; margin-top: 0; margin-bottom: 18px;">N</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;">
<strong style="color: #5A3EED;">Node</strong> — A point in a circuit where two or more components connect. All points joined by ideal wire form a single node.
</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;">
<strong style="color: #5A3EED;">Node Voltage Method</strong> — A systematic circuit analysis technique that defines voltages at each non-reference node and writes KCL equations, requiring \(n-1\) equations for \(n\) nodes.
</p>

</div>

<div style="background: #EEF4FF; border: 2px solid #A8C8FF; border-radius: 12px; padding: 28px 28px 12px 28px; margin: 1.5rem 0; box-shadow: 0 2px 8px rgba(90,61,237,0.07);">

<p style="color: #1565C0; font-weight: 700; font-size: 1.08rem; margin-top: 0; margin-bottom: 18px;">R</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;">
<strong style="color: #5A3EED;">Reference Node</strong> — The node assigned a voltage of 0 V, against which all other node voltages are measured. Also called the ground node or datum node.
</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;">
<strong style="color: #5A3EED;">Resistance Matrix</strong> — A symmetric matrix \(\mathbf{R}\) used in the mesh current method where diagonal entries are the sum of resistances in each mesh, and off-diagonal entries are the negatives of shared resistances.
</p>

</div>

<div style="background: #EEF4FF; border: 2px solid #A8C8FF; border-radius: 12px; padding: 28px 28px 12px 28px; margin: 1.5rem 0; box-shadow: 0 2px 8px rgba(90,61,237,0.07);">

<p style="color: #1565C0; font-weight: 700; font-size: 1.08rem; margin-top: 0; margin-bottom: 18px;">S</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;">
<strong style="color: #5A3EED;">Supermesh</strong> — A combined loop formed when a current source is shared between two meshes, allowing KVL to be written around the outer boundary while using the current source value as a constraint equation.
</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;">
<strong style="color: #5A3EED;">Supernode</strong> — A combined region formed when a voltage source connects two non-reference nodes, allowing KCL to be applied to the boundary while using the voltage source value as a constraint equation.
</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;">
<strong style="color: #5A3EED;">Superposition Principle</strong> — The principle that in a linear circuit, the response at any point equals the algebraic sum of responses due to each independent source acting alone, with all other independent sources deactivated.
</p>

</div>

<div style="background: #EEF4FF; border: 2px solid #A8C8FF; border-radius: 12px; padding: 28px 28px 12px 28px; margin: 1.5rem 0; box-shadow: 0 2px 8px rgba(90,61,237,0.07);">

<p style="color: #1565C0; font-weight: 700; font-size: 1.08rem; margin-top: 0; margin-bottom: 18px;">W</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;">
<strong style="color: #5A3EED;">Wye Configuration (Y)</strong> — A three-terminal resistor arrangement where three resistors share a common central node, with each resistor extending to one of the three external terminals. Also called a T configuration.
</p>

</div>
